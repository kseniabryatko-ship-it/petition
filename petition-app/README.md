# 🎖️ Петиция — Next.js приложение

## Быстрый старт

```bash
# 1. Установить зависимости
npm install

# 2. Запустить dev-сервер
npm run dev
```

Открыть: http://localhost:3000

## Страницы

| URL | Описание |
|-----|----------|
| `/` | Публичная страница петиции |
| `/admin` | Панель администратора |

## Вход в админку

**Пароль по умолчанию:** `"password"`

## Функционал

### Публичный сайт (`/`)
- Отображение заголовка, фото и текста петиции
- Форма: имя, фамилия, номер телефона
- Счётчик подписей с прогресс-баром
- После подписи — опциональный редирект на внешний URL

### Админ-панель (`/admin`)
- 📋 **Подписи** — таблица всех подписантов (имя, фамилия, телефон, дата)
- ⚙️ **Настройки:**
  - Редактировать заголовок петиции
  - Загрузить фото (показывается в начале текста)
  - Редактировать текст петиции (`**жирный**`)
  - Изменить цель по подписям
  - Задать URL редиректа после подписи

## База данных

SQLite, файл `data/petition.db` — создаётся автоматически при первом запуске.

## Production

```bash
npm run build
npm start
```
